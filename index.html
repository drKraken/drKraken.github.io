Journey Into the Wonderful World of CG

I’m not a really pro of CG, but in this article i want to tell story about my way of diving into computer graphic world. Not so long ago, i was delighted by arts of some CG artist on ArtStation and i decided to learn more as i can about 3d. So i started to search forums how technical person can dive in artistic world. So this is what i came to.

Hand drawing will be useful

First of all, you must start to train you hand drawing skills. Of course, you can use polygon modelling technics and not use software like Zbrush or Mudbox but in some period you will come to point were free hand drawing is critical.

Decide what you want to do

Then when you have some classical artistic skills you can decide what you want to learn 3d, 2d or maybe VFX . You can model characters for games of films or maybe draw concepts for it. So you must make a decision. CG world is wide and you can not learn everything for a first time.

Find goodies for begginers

Then when you make it you can start learning special software and dive into useful technics. Lets discuss some of this.

For 3D

You can use Maya, 3Ds Max, Blender, Cinema 4d , AutoCAD . As you see a lot of software and it is not end, but you must choose one you like. I reccomend Maya and Blender because i work with it and they are very powerful.
Education resources: Digital-tutors, Gnomon, Eat3d, Lynda.com, CGCookie

For 2D

Most of all time you will use Photoshop and of course you need some painting board like Wacom Bamboo because mouse painting is uncomfortable.
Education resources: Digital-tutors, Gnomon, Udemy, Matt Kohr

For VFX

Same software as for 3d and 2d. For VFX development you need to know 3d as well as 2d and have nice math skills and maybe programming for developing useful plugins. I believe, that Maya use Python for it.
Education resources: Digital-tutors, Gnomon

Of course its only top of the iceberg, but it can be useful for you for the first time and can eliminate some of the questions that I had had.

Good luck with your trip to amazing artistic world!
